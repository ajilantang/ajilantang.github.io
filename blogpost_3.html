Pada kesempatan kali ini, Saya mempelajari tentang DOM yang berguna untuk menjalankan program yang bisa langsung berinteraksi dengan user. Sehingga program JS bisa langsung ditampilkan di html, ketika user melakukan aksi pada beberpa elemen di html kita seperti klik ataupun menekan tombol tertentu. Pada pembuatan game pun bisa menggunakannya karena bisa berinteraksi dengan user.dalam melakukanya karena Js merupakan object maka Dom bisa mengambil beberapa inputan dari luar , maka pemasangan ID dan pemahaman tentang node seperti parent , child dan yang lain bisa kita pakai . Bahkan kita bisa menginputkan berdasarkan array adapula event untuk pengingat pada browser ketika suatu tindakan dilakukan oleh user seperti klik .
