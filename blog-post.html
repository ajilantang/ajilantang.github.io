blog post ini bertujuan mereview pembelajaran saya sepekean mengenai pengertian dasar-dasar pembuatan design web

Selama sepekena ini dipadatkan oleh banyak materi tentang design web sampai bagaimana perananan design web pada ssebuah web. tidak hanya mementingkan sebuah konten yang berkualitas , tetapi web haruslah memiliki design yang baik . Tujuan dari sebuah web adalah user, sehingga ketika design dari web tidak membuat user nyaman maka web itu sendiri akan mati. pembuatan web minimal menginformasikan tujuan kita kepada user. akan tetapi ketika tidak ada user yang merasa nyaman dengan web kita maka tujuan kita haruslah dipukul mundur.
